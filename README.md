CvApp
Skolövning i ASP.NET Core MVC. En enkel CV-portal där kandidater laddar
upp sitt CV och sina uppgifter, och man kan söka kandidater på skills.
Vad övningen täckte

ASP.NET Core MVC med Controllers och Views
Entity Framework Core mot SQL Server
Filuppladdning (CV-filen)
Två varianter av lagring: lokalt på disk och i Azure Blob Storage
(se CandidateService.cs vs CandidateService_Azure.cs)
Bootstrap för UI

Stack
.NET 10, EF Core, SQL Server, Azure Blob Storage, Bootstrap.
